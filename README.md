# AspNetCoreCleanArchitecture
Asp.Net Core Clean Architecture

# Prerequistes
1. .Net Core 3
2. EntityFramework Core 3
2. SQL Server

# Configuration
1. Configure database connection string in `appsetting.*.json`

# Run locally
1. `dotnet restore`
2. 'dotnet watch run`

# Testing
1. Go to [Swagger UI](https://localhost:5001/swagger/index.html)
